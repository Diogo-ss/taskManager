# TaskManager, kanban in the terminal

### Dependencies

- g++
- make
- ncurses library

## Installation

- Clone
```sh
git clone https://github.com/jaumdev1/taskManager.git
cd taskManager
```

- Compile
```sh
make
```

- Run
```sh
./grello
```
